
# Books Scraper Project

## Project Overview
This project scrapes book data from Books to Scrape website using Python.

## Target Website
https://books.toscrape.com/

## Data Extracted
- Book Title  
- Price  
- Rating  

## Features
- Pagination handling (5 pages)
- Error handling for requests
- Clean CSV output

## Tools Used
Python, requests, BeautifulSoup, pandas

## How to Run
pip install -r requirements.txt  
python scraper.py
